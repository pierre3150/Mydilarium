Cours de Développement - Programmation Orientée Objet (POO)

Concepts de Base

La programmation orientée objet (POO) est une méthode de programmation qui permet de structurer le code en objets. Voici les concepts fondamentaux :

  • Objet : Une instance d'une classe qui regroupe des données et des méthodes.
  • Classe : Un modèle définissant les attributs et comportements des objets.
  • Attribut : Une variable stockant des informations spécifiques à un objet.
  • Méthode : Une fonction liée à un objet, qui permet de manipuler ses attributs.
  • Encapsulation : Mécanisme de protection des données d'un objet en les rendant accessibles uniquement par des méthodes définies.
  • Héritage : Un mécanisme permettant à une classe de dériver d'une autre classe et de réutiliser son code.
  • Polymorphisme : Capacité de différentes classes à redéfinir une même méthode.

Exemple de Classe en Python

Voici un exemple simple d'une classe Voiture en Python :

class Voiture:
    def __init__(self, marque, vitesse):
        self.marque = marque
        self.vitesse = vitesse

    def accelerer(self, ajout_vitesse):
        self.vitesse += ajout_vitesse
        print(f"La voiture {self.marque} roule à {self.vitesse} km/h")

ma_voiture = Voiture("Peugeot", 100)
ma_voiture.accelerer(20)

Dans cet exemple, la classe Voiture possède deux attributs (marque et vitesse) et une méthode accelerer pour augmenter la vitesse de la voiture.
