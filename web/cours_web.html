Introduction au Web

Le Web

Le Web, ou World Wide Web, est un système d'information qui permet l'accès à des documents et des ressources via Internet. Ces documents sont généralement formatés en HTML (HyperText Markup Language) et peuvent inclure des éléments multimédias comme des images et des vidéos.

Langages de Base

  • HTML : Structure le contenu des pages web.
  • CSS : Stylise l'apparence des pages web, comme les couleurs, les polices et les dispositions.
  • JavaScript : Ajoute de l'interactivité et des comportements dynamiques aux pages web.

Fonctionnement

Les pages web sont servies par des serveurs web et affichées dans des navigateurs. Lorsqu'un utilisateur entre une URL, le navigateur envoie une requête au serveur, qui renvoie la page web correspondante.

Développement Web

Le développement web implique la création de sites et d'applications web. Cela inclut la conception (UI/UX), le développement front-end (ce que l'utilisateur voit) et le développement back-end (serveurs et bases de données).

Outils et Technologies

  • Éditeurs de code : Visual Studio Code, Sublime Text.
  • Frameworks : Bootstrap pour le design, React ou Vue.js pour des applications interactives.
